# ordo-website

[![Netlify Status](https://api.netlify.com/api/v1/badges/7eb8f4ff-aaa5-4316-8bfe-ad65aa55b92e/deploy-status)](https://app.netlify.com/projects/ordo-website/deploys)

Source for the Ordo marketing site, served at `ordo.ephyrasoftware.com`.

This site is built with [Astro](https://astro.build/) and deployed by [Netlify](https://www.netlify.com/) on every push to `main`.

## Develop locally

```sh
npm install
npm run dev
```

The dev server runs at `http://localhost:4321`.

To produce a production build:

```sh
npm run build
```

The output is written to `dist/`.
